机器学习之matplotlib

Date 周六 06 十月 2018 By 浮绿水 Category 技术. Tags pelican publishing

matplotlib得使用

  • 在绘图时必须进行导包

##### from matplotlib import pyplot as plt

  • 如果你要在里面添加中文· 也要导包 并进行设置：

    import matplotlib
    matplotlib.rcParams['font.sans-serif']=['SimHei'] #设置中文

  • 柱状图

  • plt.bar()

    • plt.bar(range(5),[100,200,300,400,500],label='people')

    range（设置x轴上数据的个数，里面的值和你后面设置的数值个数必须一样）

    label()进行绘制图例

    ###### 在绘制图例时 必须写 plt.legend()来进行绘制图例

    • plt.legend() #绘制图例
    • plt.xlabel('x') #给x轴进行标记（取名）
    • plt.xlabel('y') *#给y轴进行标记（取名）
    • plt.xticks(range(5),['A','B','C','D','E']) #x轴刻度
    • plt.show() #进行查看

  • 折线图

  • plt.plot()

    • plt.plot([100,50,300,200,500],color='red',label='爸爸')

    ##### lable设置图例 (不会看柱状图)

  • 扇形图（饼状图）

  • plt.pie(autopct='%1.0f%%') （在写扇形图时 必须设置百分比）
    • 举例

    plt.pie([100,200,50,150],autopct='%1.0f%%',labels=['a','b','c','d'])

    ###### labels给每个对应的板块取名

    ###### autopct设置百分比

  • 直方图（统计出现的次数）

  • plt.hist()
    • 举例

    ##### plt.hist([100,200,300,250,100])
