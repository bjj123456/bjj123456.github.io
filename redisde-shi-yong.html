Date 周二 14 八月 2018 By 浮绿水 Category 技术. Tags pelican publishing

redis的使用

  • redis对key（键）的使用

  • 执行redis的命令：redis-cli

  • set key 创建键（key）
  • keys * 查看键有哪些 keys 查看所有符合条件的key值
  • get key 查看键值的内容
  • del key 删除键
  • move key db 将当前数据库得key移动导给定得数据库db当中
  • randomkey 在数据库中随机返回一个key键值
  • rename key newkey 修改key得名称

  • type key 返回key所储存得值得类型

  • 对键值时间得操作

  • setex key time value 创建键值并添加过期时间 值

  • explre key 给指定的键值设置过期时间
  • pexpireat key 设置过期时间 以毫秒为单位设施时间戳
  • persist key 移除key值得过期时间
  • pttl key 以毫秒为单位返回key得剩余时间
  • ttl key 以秒为单位 返回给定key的剩余生存时间
